Difference b/w HTML Tags and Attributes

S.No HTML Tag HTML Attribute
  HTML tags are used to hold the elements HTML attributes are used to describe the characteristic of an HTML element in details
  HTML tags starts with "<" and ends with ">" HTML attributes are found only in starting tag
  HTML tags are almost like keywords where every single tag has unique meaning HTML attributes specify various additional properties to the existing HTML elements
